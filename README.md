# 🏥 MediSysManager

MediSysManager is a **Java-based Hospital Management System (HMS)** built as a console application.  
It allows managing physicians, patients, medicines, laboratory services, amenities, and employees efficiently.

---

## ✨ Features
- 👨‍⚕️ Manage Physicians (add, list with details like specialty, shift, degree, room)
- 🧑‍🤝‍🧑 Manage Patients (store ailment, gender, admission status, age)
- 💊 Pharmacy Management (store medicines, brand, expiry, stock)
- 🔬 Laboratory Services (add and list lab tests with fees)
- 🏥 Amenities (ambulance, canteen, admit facility, etc.)
- 👷 Employees (nurses, workers, security with salary info)
- 📅 Displays current date and time on startup
- 📂 Organized into multiple classes for maintainability

---

## ⚙️ Technologies Used
- **Java (JDK 8 or above)**
- **OOP concepts** (Encapsulation, Classes & Objects)
- **Collections Framework** (`ArrayList`, `Scanner`)
- **Console-based UI**

---

## 🚀 Getting Started

### 1. Clone the Repository
git clone https://github.com/hamzasiddiqui-coder/MediSysManager.git
cd MediSysManager
### 2. Compile the Program

javac MediSysManager.java
### 3. Run the Program
java MediSysManager
